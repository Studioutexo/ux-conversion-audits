# UX Conversion Audits

This repository contains UX and conversion breakdowns for local service-based businesses.

The goal is to identify why websites lose customers and how to improve conversion rates through simple, practical changes.

---

## What is a UX audit?

A UX audit is a structured review of a website to identify:

- User experience issues
- Conversion blockers
- Poor messaging or structure
- Mobile usability problems
- Missing calls-to-action

---

## Who this is for

We focus on UK local service businesses such as:

- Gyms
- Restaurants
- Trade services (plumbers, electricians, builders)
- Local shops and service providers

---

## Common problems found

Most websites we review suffer from:

- No clear value proposition
- Confusing layout
- Weak or missing calls-to-action
- Poor mobile optimisation
- No structured lead flow

---

## Example audit structure

1. First impression analysis  
2. Clarity of message  
3. Navigation and UX flow  
4. Conversion points (calls, forms, bookings)  
5. Mobile experience  
6. Recommendations

---

## Outcome

The goal is to turn a website from:

"visitors browse and leave"

into:

"visitors understand, trust, and take action"
